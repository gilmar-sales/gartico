# Gartico

A gartic clone made with flask + socketio

## Installing modules

Using Python 3.8

```
    pip install -U flask
    pip install -U flask_socketio
    pip install -U mysql-connector
```

## Windows init

```
set FLASK_APP=index.py

flask run
```

## Linux/Mac init

```
export FLASK_APP=index.py

flask run
```
